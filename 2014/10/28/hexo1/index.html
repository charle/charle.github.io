Hexo常用命令

到这里，我们已经了解了如何配置一个hexo博客，并且通过测试

hexo n ‘name’ #建立新的博文
hexo g #hexo generate 生成对应的html,css,js文件
hexo s #hexo server 启动服务
hexo d #hexo deploy 布置到git上去

几个常见命令用来新建，更新，预览，同步到git pages博客

nodejs
hexo
