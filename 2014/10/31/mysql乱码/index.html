mysql乱码

打开终端登陆到服务器，登陆mysql数据库,查询数据总是乱码，一开始觉得是终端编码设置有问题，没放在心上，后来网上搜了些东西，原来mysql也可以设置编码格式输出的

mysql输出乱码命令：

mysql> set names ‘gbk’;如果不行则可能用 set names ‘utf8’;

php输出乱码命令

mysql_query(“set name ‘gbk’”);

phpmyadmin里乱码

phpMyAdmin的config.inc.php中有没有设置$cfg[‘DefaultCharset’]=’utf-8’;
